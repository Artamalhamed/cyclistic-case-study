# Cyclistic Case Study  
### Google Data Analytics Capstone Project

Cyclistic Case Study
How Annual Members and Casual Riders Use Bikes Differently
1. Business Task

The goal is to understand how annual members and casual riders use Cyclistic bikes differently to support marketing decisions.

2. Data Used

Twelve months of trip data including start time, end time, ride length, and user type.

3. Data Cleaning

Combined monthly files

Removed invalid or missing rows

Created ride_length

Added day_of_week

Standardized column names

4. Analysis Summary

Members ride more often and mainly on weekdays

Casual riders take longer trips and ride more on weekends

Seasonal patterns show higher casual usage in warm months

5. Key Insights

Casual riders = long rides

Members = short, frequent rides

Weekend vs weekday usage differs

Summer increases casual ridership

6. Recommendations

Offer summer membership promotions

Highlight cost savings of annual membership

Target weekend riders through digital ads

7. Conclusion

Members and casual riders show different patterns. These insights can guide strategies to convert casual riders into annual members.
